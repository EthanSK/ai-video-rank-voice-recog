Quick Setup Guide

1

Install Prerequisites

Set up the required software on your system:

# Create Python environment python3 -m venv venv source venv/bin/activate # Install RealtimeSTT pip install RealtimeSTT # Install Node.js dependencies npm install
2

Load Chrome Extension

Add the extension to your Chrome browser:

1. Open chrome://extensions/ 2. Enable "Developer mode" 3. Click "Load unpacked" 4. Select chrome-extension/ folder 5. Verify "Arena Helper" appears
3

Start the System

Launch both the main app and voice recognition:

# Terminal 1: Main application npm run dev # Terminal 2: Voice recognition ./start-voice.sh
4

Navigate & Vote

Go to the arena and start ranking:

1. Visit artificialanalysis.ai/text-to-video/arena 2. Solve Cloudflare once (if needed) 3. Two video windows open automatically 4. Say "up" or "down" to vote 5. Daniel announces the model name
